# WealthCopy 90초 데모 스크립트

목표 러닝타임: **1분 30초**. 미리 채워진 샘플 프로필과 균형형 경로를 사용하면 전체 흐름을 안정적으로 재현할 수 있다.

## 0:00–0:12 — 문제와 한 문장 가치

화면: 랜딩의 `L6 → L7` 경로 그래픽과 핵심 문구.

멘트: “자산관리 앱은 지금 잔고는 보여 주지만, 다음 단계로 가는 실행 경로는 직접 설계해야 합니다. WealthCopy는 나와 비슷한 조건의 대표 경로를 비교하고 월간 계획으로 복사합니다.”

동작: `내 조건 직접 입력`을 클릭해 입력 영역으로 이동한다.

## 0:12–0:26 — 조건 입력

화면: 미리 채워진 샘플 프로필.

멘트: “현재 L6, 목표 L7, 월소득 650만 원, 가용액 310만 원, 부채 비율 18%, 비상자금 5개월입니다. ‘내년에 이사 가능성이 있어 현금 여유를 유지하고 싶다’는 자연어 제약도 함께 봅니다.”

동작: 값을 빠르게 훑은 뒤 `경로 3개 비교하기`를 클릭한다. 입력 단계를 생략하는 짧은 시연에서는 랜딩의 `샘플로 바로 비교`를 사용한다.

## 0:26–0:48 — 세 경로 비교와 GPT‑5.6

화면: 안정형·균형형·빠른형 카드, 상단 출처 배지, 요약 설명.

멘트: “금액과 기간은 규칙 엔진이 같은 기준으로 계산합니다. GPT‑5.6은 이 숫자를 바꾸지 않고, 이사와 현금 여유 같은 제약을 해석해 적합 이유와 트레이드오프를 Structured Output으로 반환합니다. 지금은 균형형을 먼저 비교하라고 설명합니다.”

동작: 안정형 6.4년·월 240만 원, 균형형 5.2년·월 310만 원, 빠른형 4.5년·월 420만 원을 짧게 훑는다. 빠른형 버튼이 가용액 부족으로 비활성화된 것도 가리킨다.

## 0:48–1:05 — 경로 복사

화면: 균형형 카드.

멘트: “추천을 자동 실행하지는 않습니다. 사용자가 감당 가능한 경로를 직접 고릅니다.”

동작: 균형형의 `이 경로를 계획에 복사`를 클릭한다. 확인 모달에서 “복사는 매매나 자동이체가 아니라 체크리스트 생성”이고 입력값과 체크 상태가 이 브라우저에 저장된다는 문구를 보여 준다. 교육용 시뮬레이션 확인란을 먼저 선택해 버튼이 활성화되는 것을 보여 주고 `계획에 복사`를 클릭한다.

## 1:05–1:20 — 매월 실행

화면: 월간 체크리스트와 0% 진행률.

멘트: “선택한 경로는 현금 여유, 부채 일정, 월 실행 한도, 월말 재점검의 네 가지 행동으로 바뀝니다. 검증된 입력과 선택, 체크 상태는 이 브라우저에만 저장되어 다음 방문에 복원됩니다.”

동작: 첫 체크리스트를 완료해 진행률이 25%로 바뀌는 것을 보여 준다. 기본으로 꺼진 `리마인더 화면 데모` 스위치를 가리키며 실제 알림이나 예약은 만들지 않는다고 말한다.

## 1:20–1:30 — 안전성과 마무리

화면: 계획 화면의 `GPT‑5.6` 배지와 교육용 시뮬레이션 안내.

멘트: “개인정보 형태와 금융 실행 요청은 모델 전에 차단하고, AI가 새 수익률이나 상품을 만들 수 없도록 스키마와 의미 검증을 적용했습니다. 키나 네트워크가 없어도 fallback이 유지됩니다. WealthCopy는 자산이 아니라 실행 경로를 복사합니다.”

## 데모 전 준비

- [ ] `C:\wealth_copy_openai_challenge`에서 `pnpm.cmd check` 통과
- [ ] `pnpm.cmd dev` 실행 후 `http://localhost:3000` 접속
- [ ] 라이브 AI 시연 시 `.env.local`의 `OPENAI_API_KEY`와 `OPENAI_MODEL=gpt-5.6` 확인
- [ ] `GPT‑5.6 분석` 배지가 표시되는지 사전 확인
- [ ] fallback 시연 시 키를 비우고 `규칙 기반 설명` 배지와 경고 확인
- [ ] 반복 녹화 전 `localStorage`의 `wealthcopy-demo-plan-v1`을 지우고 새로고침해 저장 계획과 진행률을 초기화
- [ ] 요청을 반복 테스트했다면 분당 세션 제한을 피하도록 60초 기다리거나 `wealthcopy-anonymous-session`도 지우고 새 세션으로 시작
- [ ] API 키, 개발자도구, 개인 데이터, 다른 탭이 화면에 노출되지 않도록 정리
- [ ] 캡션과 음성 확인, 최종 영상 1분 30초 이내

## 라이브 API 실패 시 복구 멘트

“지금은 API 연결 대신 동일한 규칙 엔진의 fallback 설명이 표시됐습니다. 경로 비교와 복사 흐름은 그대로 유지되고, 화면 배지가 결과 출처를 명확히 보여 줍니다.”
